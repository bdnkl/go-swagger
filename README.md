# go-swagger

Go api with swagger documentation

## Install swag

```
go install github.com/swaggo/swag/cmd/swag
```

## Create swagger docu automatically

```
swag init
```

## Build server

```
go build main.go
```

## Run server

### Windows

```
main.exe
```

## Show documentation

Go to http://localhost:8080/swagger/index.html to show swagger documentation.
